Welcome to Icer.

Icer is an online, cross-browser compatible, 3d music streaming website. You can create custom playlists using your favourite songs or discover new artists or songs from your playback hostory. The project is completely open source, you can clone it using the following command: 


    $ git clone https://github.com/colin-dumitru/icer.git
