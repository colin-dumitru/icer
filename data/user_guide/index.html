Navigation

The user is able to navigate to one of the five sections of the application from the vertical menu bar located in the left side of the page, either by clicking on a menu item or by dragging the menu selector to the desired section.

  • Search

    • In this section of the application, the user has the possibility to search for anything music-related by providing the desired input. According to what was searched and the selected tab, connections with songs by artist, album or genre will be displayed and the user will be able to play the song, create a new search session or add the selected song to playlist.

  • Playlist

    • In the Playlist section, the user is able to manage his playlists, create new ones and remove individual songs or the entire playlist. The Share option allows active social media users to share their musical taste with friends and followers on Facebook, Twitter or Google+.

  • Radio

    • The Radio feature of the application allows the user to choose the criteria for the automatically generated radio. He can select one or multiple default criteria or provide input for the custom search option and a list of songs will be provided for him.

  • History

    • In the History section, the user has the possibility to monitor his history via a timeline graph, which displays a top of the most played genres and artists from a selected period of time.

  • Chart

    • This section of the application displays a chart of the most listened to songs by year, month, week or day, along with informative details on the song's current state such as weeks on chart and peek position.

Search

Fig 2.1

Adding a new search session

Fig 2.2
  • Open the item list by hovering over the item bar on the right side of the main content area.
  • Type the desired query in the input box.
  • A new search session based on the provided input is added in the item list and displayed.

Browsing search sessions

  • The search sessions can be accessed either by selecting them from the item list displayed by hovering over the item bar on the right side of the main content area or by using the up and down keyboard arrows.

Deleting a search session

  • A search session can be deleted by clicking on the icon situated on the right side of the search categories menu. (see Fig 2.1)

Browsing search results

Fig 2.3
  • The search results are grouped in 4 categories: Songs, Artist, Albums and Genre.
  • Browse through the categories by either selecting the corresponding top button or by using the left and right keyboard arrows.
  • The Songs category contains a primary group of songs, which have the searched input included in their title. The group is always visible in this search category. Each element of the previous group has a list of similar songs attached. These lists are initially hidden, but become visible in the right area next to the correspondent primary song when hovering over that song. Another way of making them visible is by hovering over the section reserved for their placement.
  • The Artist category contains a group of artists, whose names identically or partially match the searched input. The resulted list of artists is always visible. Similarly to the "Songs" category, each artist has an associated group of songs, group that becomes visible when hovering actions like the ones explained in the previous category are performed.
  • The Albums category contains the list of albums for which the searched input is a part of their title. Each albums has a list of songs, presenting the same properties described for the previous lists of songs.
  • The Genre category contains the genres that have matched the searched input or parts of it. Each genre is followed by a list of songs pertaining to that genre. Unlike the previous list of songs, the songs in this category are always visible.
  • More category results can be viewed by clicking the See More button at the bottom of the vertically scrolling window.
  • More songs can be viewed by clicking the + button at the right side of the horizontally scrolling panel.

Viewing song options and artist info

Fig 2.4
  • By clicking on a song's cover art a new window opens, containing information about the artist (image, biography, touring status) and a list of available options (Play Now, Add To Playlist, Add To Now Playing, Search From Here).
  • The window will disappear by moving the cursor outside of it.

Playing a song

  • Select the Play Now option from the song options window. (see Fig 2.4)
  • The selected song will be added to the now playing list and playback will automatically begin.

Adding a song to a playlist

  • Select the Add To Playlist option from the song options window. (see Fig 2.4)
  • A drop-down list containing the existing playlists and the option to create a new one will be displayed.
  • Type the name of a new playlist in which to add the song or alternatively add the song to an existing one.

Adding a song to the music player

  • Select the Add To Now Playing option from the song options window. (see Fig 2.4)
  • The selected song will be added to the now playing list.

Searching for similar songs

  • Select the Search From Here option from the song options window. (see Fig 2.4)
  • A new search session will be created and displayed.

Playlist

Fig 3.1

Creating a new playlist

Fig 3.2
  • Open the item menu by hovering over the vertical bar on the right side of the playlist section.
  • Provide name input for the new playlist in the sidebar item list by typing it in the text box.
  • A new playlist will be created with the specified name and added to the item list.

Selecting a playlist

  • View previously created playlists by hovering over the vertical bar on the right side of the playlist section. (see Fig 3.2)
  • The playlists can be viewed by either using the up and down arrow or by selecting it from the item list from the sidebar.

Playing a playlist

  • Select the Play button from the upper-right corner of the main content area.
  • The songs from the playlist will automatically start playing.

Sharing a playlist

Fig 3.3
  • Select the Share button from the bottom-right corner of the main content area.
  • Share the current playlist on Facebook, Twitter or Google+.
  • On choosing the Share option, a new window is displayed in which the user can login to one of the previously mentioned social media websites and share the playlist.

Deleting a playlist

  • Select the icon.
  • The current playlist will be deleted.

Playing selected song from playlist

Fig 3.4
  • Select a song from the playlist.
  • A menu with three options will appear in the right side of the song image.
  • Select the Play Now option.
  • The track will automatically start playing.

Searching similar songs/artists

  • Select a song from the playlist. (see Fig 3.4)
  • A menu with three options will appear in the right side of the song image.
  • Select the Search From Here option.
  • The action will redirect the user to the Search part of the application, where similar songs to the one selected will be displayed.

Removing a song from playlist

  • Select a song from the playlist. (see Fig 3.4)
  • A menu with three options will appear in the right side of the song image.
  • Select the Remove From Playlist option.
  • The song will be removed from the playlist.

Radio

Fig 4.1

Selecting a default criterion

Fig 4.2
  • Select the Add Criteria option.
  • The default criteria are Recent Songs, Recent Genres and Recent Albums.
  • Select one or multiple criteria from the ones provided.
  • The criterion will be added to the selected criteria list.

Adding a custom criterion

  • Provide input for the Custom Search option. (see Fig 4.2)
  • When pressing the Enter key, the custom search input will be added to the selected criteria list.

Playing the radio

  • Select the Play button. (see Fig 4.1)
  • Songs are generated from the selected criteria and added to the player.

Clearing selected criteria

  • Select the Clear All option. (see Fig 4.1)
  • All selected criteria will be deleted.

Resetting the radio

  • Select the Reset option. (see Fig 4.1)
  • Default options will be added to the selected criteria list.

Deleting a criterion

Fig 4.3
  • Select the X option next to the selected criterion.
  • The criterion will be deleted from the selected criteria list.

History

Fig 5.1

Selecting a history point

  • Drag the selection slider to the desired position.
  • The selected week interval will be displayed at the top of the page.
  • The bar chart will be updated with the 4 most listened to genres from the selected week and also the 4 most listened to artists will be displayed.

Displaying a chart

Fig 6.2
Fig 6.3
  • Select the When? option.
  • A new window in which a year can be selected is displayed.
  • Either select the Cancel option to return to the previously generated chart or select a year to proceed to the next selection window.
  • By choosing a year, a new window in which a month can be selected is displayed.
  • Selecting the Done option generates the chart for the desired year, selecting the Cancel button returns to the previously generated chart, while selecting a month displays the next selection window.
  • If a month is selected, a new window containing the weeks is displayed.
  • Either click the Done button in order to generate the chart for the desired month, click the Cancel button to return to the previously generated chart or select a week to proceed to the next selection window.
  • By selecting a week, a new window in which a specific day can be selected is displayed.
  • Press the Done button to generate the chart for the selected week, press the Cancel button to return to the previously generated chart or select a specific day for which to generate the chart.
  • For each song, the title, artist, album cover, together with the position change from the previous time span (day, week, month, year) represented as / / , as well as the peek position () and number of weeks in top () will be displayed.

Searching for similar songs

Music Player

Fig 7.1

The music player displays the list of currently playing songs and can be viewed by hovering over the bottom of the page at each side of the playback container.

Arrange songs

  • The songs in the music player can be rearranged by clicking and dragging them to the desired location.

Delete a song

Fig 7.2
  • Access the song menu by clicking and holding on the desired song.
  • The song menu with Delete and Delete All options will appear.
  • Select the Delete option from the menu.
  • The song will be removed from the music player

Delete all songs

  • Access the song menu by clicking and holding on the desired song. (see Fig 7.2)
  • The song menu with Delete and Delete All options will appear.
  • Select the Delete All option from the menu.
  • All songs will be removed from the music player
